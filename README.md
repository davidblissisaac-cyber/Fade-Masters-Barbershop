# Fade Masters — Barbershop Website

A premium, responsive barbershop website built with vanilla HTML, CSS, and JavaScript — no frameworks.

**Live site:** https://fademastersbarbershop.netlify.app

## Features
- Mobile-first responsive design
- Sections: Home, About, Services, Pricing, Gallery, Testimonials, FAQ, Contact
- Client-side booking form with validation
- FAQ accordion, scroll-reveal animations, sticky nav
- Google Maps placeholder ready for embed
- Respects `prefers-reduced-motion` for accessibility

## Design concept
The visual identity is built around the shop's name: the "fade" concept
shows up in the gradient headlines, section transitions, and a
ticket-styled pricing menu instead of a plain price list.

## Tech
HTML5, CSS3 (custom properties, no framework), vanilla JavaScript.
